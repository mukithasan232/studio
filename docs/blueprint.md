# **App Name**: Affiliate AI Automator

## Core Features:

- Affiliate Link Input: Allow users to input an affiliate product URL for processing.
- AI-Powered Product Data Generation: Automatically generate product name, description and other relevant data from the provided URL using AI tool.
- Product Preview: Display a preview of the AI-generated product data, allowing users to review before publishing.
- Firestore Integration: Store and retrieve product data in real-time using Firestore database.
- Real-time Product List: Display a live, updating list of products stored in the Firestore database.
- Data scraping: Extract the product name, the price and the image url from the linked website.

## Style Guidelines:

- Primary color: Indigo (#4F46E5) to align with the brand color, and communicate professionalism
- Background color: Very light indigo (#F0F1F9), similar hue to primary color but desaturated to 20% for a clean look
- Accent color: Sky Blue (#7DADEF), providing contrast while complementing the indigo tones
- Body and headline font: 'Inter', a sans-serif font with a modern, neutral look for both headlines and body text
- Use clear and professional icons, related to e-commerce and data import, in a consistent style
- Maintain a clean, organized layout with clear sections for input, preview, and database display
- Subtle loading animations when generating data
